Core concepts

Classical encryption principles

Classical ciphers operate on letters without computers. Substitution (mapping letters) and transposition (rearranging) are the pillars. Monoalphabetic replaces each letter with a fixed substitute. Playfair encrypts digraphs to hide frequency. Hill uses linear algebra (matrix multiplication mod 26).

Encryption strength: Kerckhoffs's principle — only key secrecy matters.
Monoalphabetic

Substitution cipher · 1:1 mapping

Each plaintext letter maps to a unique ciphertext letter. The key is a full permutation of the alphabet. Example: A→M, B→N,... (ROT13 is a special case).

Plain: HELLO WORLD
Cipher: IFMMP XPSME (shift +1)
Polygraphic · Playfair

Digraph substitution (5×5 grid)

Uses a keyword to fill a 5x5 square (I/J merged). Encrypts pairs: same row → shift right; same column → shift down; rectangle → swap corners.

Keyword: MONARCHY → grid built, "HE" → "EC" etc.
Linear algebra · Hill

2×2 matrix encryption mod 26

Plaintext vector [p1,p2] × key matrix K = cipher vector. Decryption uses inverse matrix K⁻¹, exists iff det(K) is coprime to 26.

K = [[3,3],[2,5]] , det=9 → invertible.
